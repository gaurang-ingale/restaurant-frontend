# Restaurant Template (Front-End)

## Description:

This is an example template for a restaurant website.

## Technologies used:

### For the front-end (this repository):

- NextJS
- SASS
- HTML5
- MdxJS (next-mdx-remote, for parsing mdx i.e. markdown and jsx as props)

### For the back-end:

1. CMS
   - Strapi CMS
2. Database
   - MariaDB
